Sistemas Operativos

Edt (VMS)


    



  1. CARACTERISTICAS GENERALES
    2. 

     El editor por defecto del sistema VMS es EDT.

  Este editor puede trabajar en modo linea (el prompt es un "*"), o en modo
  pantalla, en este caso el teclado esta definido para realizar una serie
  de funciones. El teclado que se utiliza como base es el de una VT200,
  este editor utiliza la parte numerica y las teclas de cursor.

  Cada tecla realiza una funcion concreta Insertar, Borrar (caracteres,
  palabras, lineas), CUT, PASTE, Avanzar, Retroceder, Buscar, Sustituir.

  Cambio de modo :

		       teclas: 
		      <----------------
	       linea                     pantalla
		      ---------------->
		       comando:	CHANGE

  Todos los comandos del modo linea se pueden abreviar excepto QUIT.





  2. FUNCIONES MAS USUALES
    3. 

       
     
    • Funciones salir
      • 

      - edt ---------------------------
     |  desde modo linea
     |     exit --- sale salvando cambios
     |     quit --- sale y NO salva cambios
      ---------------------------------

     
    • Funcion refresco de pantalla
      • 

      - edt --------------------------
     |
     |       Tecla <CTRL/W> refresca la pantalla
     |
      --------------------------------

     
    • Funcion peticion de ayuda
      • 

      - edt --------------------------
     |     modo linea :
     |
     |       help
     |
     |     modo pantalla :
     |
     |       pulsar tecla PF2 (en vt100)
     |       (salir de la ayuda con barra espaciadora)
      --------------------------------  

     
    • Otras Funciones...
      • 

      - edt --------------------------
     |
     |   buffers
     |   -------
     |    Para trabajar se dispone de BUFFERS que el usuario puede
     |    crear y borrar a voluntad, se puede copiar y/o mover
     |    informacion entre los BUFFERS.
     |    Al salir del editor con EXIT el unico BUFFER que se salvara
     |    sera el MAIN correspondiente al que se crea con el fichero
     |    que se indica en el comando.
     |
      --------------------------------
