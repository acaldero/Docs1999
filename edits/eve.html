  1. CARACTERISTICAS GENERALES
    2. 

     Tambien dispone de comandos en modo pantalla o modo linea, el modo
  de trabajo por defecto es el de pantalla, este editor solo utiliza las
  teclas de cursor para trabajar (ademas de la tecla HELP y la tecla DO,
  que sirve para introducir comandos del modo linea).

  Para llamar a este editor se utiliza el comando

	     EDIT/TPU fichero

  Una de las ventajas de este editor es que esta escrito con un "editor
  de editores" denominado TPU, este programa permite al usuarios escribir
  sus propias rutinas que amplien o modifiquen la funcionalidad del editor.
  Estas rutinas se pueden asociar a nuevas teclas y/o comandos definidos
  por el usuario. Los programas escritos en TPU se pueden compilar y salvar
  generando un extension del EVE que el usuario puede utilizar cada vez que
  use el comando "EDIT/TPU". Si alguien se anima puede encontrar informacion
  en el HELP dentro del editor y tambien utilizando el comando HELP EDIT/TPU
  del sistema.

  Otra ventaja de este editor es que puede emular la funcionalidad del
  EDT en modo pantalla, para ello basta con utilizar el comando de modo
  linea SET KEYPAD EDT.
