Sistemas Operativos

Emacs (varios)


    



  1. CARACTERISTICAS GENERALES
    2. 

     El Emacs es un editor de pantalla completa. Es muy usado
  conjuntamente con Lisp.

  Como notacion para entender las teclas, hay que saber que:
       C-  --- mantener Control       apretado y presionar 
       M-  ---    "     Meta,Alt,Edit     "    "     "       "




  2. FUNCIONES MAS USUALES
    3. 

       
     
    • Funciones salir
      • 

      - emacs -------------------------
     |     C-z     --- hace un "dosshell"
     |     C-x C-c --- sale
      ---------------------------------

     
    • Funciones de Movimiento en el texto
      • 
     
        
        
      • movimientos dentro de la pantalla
        • 

          - emacs -------------------------
         |
         |      M-v --> pantalla anterior
         |
         |
         |         -->               C-p                 [linea i-1]
         |        |
         |        |
         |         --  word1 w   xC-bxxxC-fx   w  word4  [linea i  ]
         |        |
         |        |
         |         -->               C-n
         |
         |              ^    ^       |||       ^    ^
         |              |    |       |||       |    |    [linea i+1]
         |              |     ------- | -------     |
         |              |         M-b | M-f         |
         |              |             |             |
         |               ------------- -------------
         |               C-a           C-e
         |
         |      C-v --> pantalla siguiente
         |
         |      NOTAs:
         |      ------
         |       - principio oracion ... x .... final oracion
         |           M-a                         M-e
         |       - C-u 8 C-f mueve cursor 8 caracteres a delante.
         |
          ---------------------------------
          
         
      • movimiento entre pantallas
        • 

          - emacs -------------------------
         |     M-< --> principio de fichero
         |     M-v --> pantalla anterior
         |     C-v --> pantalla siguiente
         |     M-> --> final de fichero
          ---------------------------------

         
      • movimiento entre 'ventanas'
        • 

          - emacs -------------------------
         |     C-x 1  --- queda solo la ventana que tiene el cursor
         |     C-x 2  --- crea dos ventanas
         |     C-x o  --- salta a la siguiente ventana
         |     C-M-v  --- avanza una pagina en la otra ventana
          ---------------------------------
     
      

     
    • Funciones gestion ficheros
      • 

      - emacs -------------------------
     |     C-x C-f  --- Busca y edita el fichero que se le diga.
     |                  Si no existe, lo crea.
     |                  ( para pasar de un fichero a otro, hay
     |                    que "buscarlo" )
     |     C-x C-s  --- Salva a fichero
     |                  ( al original se le aniade "~" al nombre )
      -------------------------

     
    • Funciones busqueda en texto
      • 

      - emacs ------------------------
     |     C-s <str><Ret> --- busqueda hacia delante
     |         C-s para seguir buscando el mismo <str>
     |     C-r <str><Ret> ---     "      "   atras
     |         C-r para seguir buscando el mismo <str>
     |     ( NOTA: se realiza una busqueda incremental! )
      --------------------------------

     
    • Funciones reemplazamiento en texto
      • 

      - emacs ------------------------
     |     M-x "repl s<TAB>"<Return><strorg><Ret><strdest><Ret>
     |             -- replaza
      --------------------------------

     
    • Funciones insertado y borrado (y desborrado)
      • 

      - emacs ------------------------
     |
     |      <Del>     borra el caracter justo antes   del cursor
     |      C-d         "   "     "       "   despues  "    "
     |
     |      M-<Del>   mata la palabra inmediatamente antes del cursor
     |      M-d        "   "  siguiente  palabra  despues   "    "
     |
     |      C-k       mata desde la posicion del cursor hasta el final
     |                           de la linea
     |      M-k         "  hasta el final de la actual oracion
     |
     |      ( NOTA: cuando se "mata", no se puede volver a recuperar )
     |      ( NOTA: las oraciones empiezan y terminan con "." )
     |      ( NOTA: C-u 8 *  --- ******** )
     |
     |      Desborrando:
     |      C-y   --- recupera lo inmediatamente antes borrado
     |      M-y   --- cada vez que da, se visualiza todos los
     |                textos a desborrar y cuando se suelta, el
     |                elegido se queda permanente
     |      C-x u --- deshace los cambios del comando inmedia..
     |                ( ejecutar varias veces para ir recuperando )
     |      C-_   --- equivalente a C-x u
     |                ( _ puede salir en terminales dec manteniendo
     |                  apretada CTRL y tecleando / )
     |
      --------------------------------

     
    • Funcion refresco de pantalla
      • 

      - emacs ------------------------
     |     C-l --- Borra toda la pantalla y reescribe todo poniendo el
     |             texto cerca del cursor en el centro.
     |             (Es control-L, no control-1.)
     |     C-u 0 C-l --- rescribe situandose al inicio de la pantalla.
      --------------------------------

     
    • Funcion peticion de ayuda
      • 

      - emacs ------------------------
     |     C-h   --- pedir ayuda
     |               ( C-h ? --- saca un indice )
     |               ( C-h b --- lista de funciones )
      --------------------------------  

     
    • Otras Funciones...
      • 

      - emacs ------------------------
     |
     |   "buffers"
     |   ---------
     |     C-x C-b  --- Lista de buffers
     |     C-x s    --- Recorre la lista de todos los buffers
     |                  asociados a sus respectivos ficheros y
     |                  pregunta si el usuario quiere salvarlos
     |
     |   modos de funcionamiento
     |   -----------------------
     |     M-x <modo><Ret> --- cambio a otros modos de funcionamiento
     |                         ( fundamental-mode , text-mode, lisp-mode,
     |                           auto-fill-mode (4 on/off) , ... )
     |     C-u 2 0 C-x f   --- cambia el marginado a 20 columnas.
     |
     |   nivel de edicion recursivo
     |   --------------------------
     |     M-x top-level<Ret> --- sale del nvel de edicion recursivo
     |                            se detecta por que en nivel actual
     |                            viene indicado entre [,]
     |                            ( ejemplo [(fundamental)] )
     |
     |   cuando emacs parece colgado ...
     |   -------------------------------
     |     C-g --- Para cualquier funcion del emacs de una forma
     |             segura y tambien sirve para descartar argumentos
     |             numericos (cualquiera) que nos demos cuenta que
     |             hemos metido mal
     |
     |     NOTA: salir de <ESC><ESC> : con n
     |
      --------------------------------
