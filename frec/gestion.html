Comandos y programas de Gestion



Supervision Y Gestion

 

    # su  (ux)

      * Permite, dando la password, ser superusuario!
      * Si eres superusuario ya, poniendo su <usuario>, entraras
        como si fueras ese usuario y podras hacer lo que
        puede hacer el usuario. Para salir, poner exit y vuelves
        al shell del superusuario. ( el comando whoami lo usa
        normalmente el superusuario cuando no se acuerda "quien"
        es en ese momento ). Si no eres superusuario, al poner
        'su <usuario>' te pedira la password de ese usuario.

    # shutdown -r now  (ux)

      * Manda un aviso a los usuarios de que el sistema operativo
        va a resetear la maquina; despues va mandando la senyal de
        terminacion a los procesos y despues la senyal de matar
        ( para los procesos que se hallan echo el longi ); por ultimo
          resetea.

        - Hay dos comandos muy parecidos :
           - 'halt'
              no resetea, "cuelga" la maquina
           - 'reboot'
              permite mas cosas que shutdown, como resetear
              sin mandar ninguna senyal!,...

    # chgrp <nuevo grupo> <fichero>  (ux)

      * Cambia el grupo al que pertenece un fichero.

    # chown <nuevo duenyo> <fichero>  (ux)

      * Cambia el duenyo al que pertenece un fichero.
    
    # newgrp <grupo>  (ux)

      * Crea un nuevo grupo de usuarios.
        ( lo anyade al fichero /etc/group )

    # last (ux)

      * Lista los usuarios que se han conectado con la fecha y hora
        de salida del sistema y minutos que ha estado en el.
      * Para un usuario en particular, "last <user>"

    # whoami (ux)

      * Presenta que usuario eres ( a990... )

    # groups (ux)

      * Presenta a que grupo perteneces

    # id (ux)

      * Presenta el identificacion de usuario (UID) y el identi-
        ficador de grupo (GID), tanto el real como el efectivo
        ( tanto el UID como el GID son normalmente numero de 3
          cifras que identifica univocamente al usuario y al
          grupo al que pertenece )

    # tload (ux) --- indicate (cms/cp)

      * Presenta un "grafico" (con texto) de la carga del sistema
        ( si usas X-Windows, usa xload )

      * Particularidades :
