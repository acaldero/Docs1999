Solucionar tipos caracteres raros [Unix]

 

        En terminales que interpreten caracteres de escape, se pueden
     conseguir "efectos especiales", ( como animaciones e incluso musica
     a partir del pitido de alarma de los terminales de DEC ) pero
     tambien problemas pues si se "typea" un fichero binario, puede
     darse la casualidad que su contenido sea interpretado, y tenga
     caracteres de escape impredecible. Uno de los problemas mas fasti-
     diosos es cuando los caracteres son rayas! y puntos!;
     Solucion :

              - teclear ( a ciegas! ) "more /bin/sh"
                ( o bien "cat /bin/sh | more" )
              - dar a la barra espaciadora hasta que las letras
                  More y el % se vean con claridad.
              - salir del more con "q".




Paso de ficheros binarios a ascii [general]

 

        Si os traeis un fichero asc en binario hay un truco para los
     retornos de carro. Editais el fichero con el edit de DOS y dais
     a salvar; el se encarga automaticamente de convertir.




Ftps no interactivos [general]

 

         Si el ftp no es interactivo, en algunos sistemas el caracter
      @ lo interpreta como enter. Que hacer?.

      He aqui un ejemplo de un fichero .com para zipi :

               $ftp asterix.fi.upm.es
               
               user anonymous
               pass a990123@t800.fi.upm.es
               ...
               exit

      Observar que cuando el ftp te pida el username, leera un enter!.
      Para el resto de sistema, hacer lo equivalente.




En que maquina tendran los ficheros que busco ...??? [general]

 

         Simplemente usar la aplicacion de correo electronico de vuestra
      maquina y mandar un 'mail' de la siguiente forma a 'archie' :

          archie@archie.funet.fi
         SUBJECT:
         prog 
         prog 
          ........
          ......
         quit
         .

        Tarda unas de unos pocos minutos a 8 horas.
        Archie busca :
          - por patrones DISTINGUIENDO entre mayusculas y minusculas.
            por ello si por ejemplo nuestro tema es music, poner
            prog music, prog Music, prog MUSIC, ...
          - en Ingles! (don't forget that!)




Nombre ficheros Unix con caracteres "especiales" [Unix]

 

         El nombre de un fichero en Unix puede tener caracteres
      "especiales", para escribirlos, colocar antes '\' :
      ( caracteres especiales son aquellos que para el sistema
        operativo tienen un significado especial, que es lo
        primero que interpreta )

          ej.)       /directorio!&/hola

                  md /directorio\!\&/hola
                  cd /directorio\!\&/hola
